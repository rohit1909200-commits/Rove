# ✈️ ROVE Plane Shooter

A futuristic **gesture-controlled plane shooting game** built using **HTML, CSS, and JavaScript**, powered by **MediaPipe Hand Tracking**.

Control your plane using your hand gestures through the camera — no keyboard required 🚀

---

## 🎮 Features

* ✋ Hand gesture controls using camera
* ✈️ Smooth plane movement
* 🔥 Pinch gesture to shoot bullets
* 🚀 Fist gesture to activate boost
* 💥 Dynamic obstacles & collisions
* 🪙 Collect coins to increase score
* ❤️ Health system (HP-based gameplay)
* 🎯 Real-time HUD (Score, HP, Status)
* 🧠 Smart damage scaling system

---

## 🕹️ Controls

| Gesture        | Action     |
| -------------- | ---------- |
| 👆 Move finger | Move plane |
| 🤏 Pinch       | Shoot      |
| ✊ Close fist   | Boost      |

## ⚠️ Important

* Camera access is required for gameplay
* Works best on Chrome browser
* Make sure lighting is good for accurate hand tracking

## 📸 Game Logic Overview

* Real-time hand tracking using MediaPipe
* Finger tip controls plane movement
* Gesture recognition triggers actions (shoot / boost)
* Collision detection handles damage and scoring
* Smooth rendering using `requestAnimationFrame`
